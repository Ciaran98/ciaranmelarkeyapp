CS50 Online Course

Earlier this year, I enrolled in the online course CS50. This course teaches the fundamentals of computer science, which I already knew. I decided to do the course to keep myself occupied, and to keep myself up to date with computer science. I was also mainly interested in doing the practical assignments that were in the course. These assignments were a nice challenge that kept me occupied while I searched for work. In one assignment, I was tasked with applying filters over certain images. The filters were greyscale, sepia, blur, and flip. Another assignment was to recover images from an SD card. The program would read the data, find what bytes correlated to the start and end of an Image, and write the data of the image.

This website!

I began working on this website because I believed that it would help me in finding employment. The website began as a simple, static website hosted on GitHub pages that used HTML, CSS, and JavaScript. I had set a simple goal for myself: create a nice looking page that would allow me to show off my talents. The first iteration of the website was entirely front-end, and completely static. The most complex aspect of the website was the Code Snippet section. This used a JavaScript function to reveal a hidden div and play a smooth opening animation. The text within was hardcoded into the HTML page. Once I was finished working on this iteration of the website, I knew I could improve it.
Unfortunately, to improve the site and make it more dynamic, I would have to search for different hosting, as GitHub pages are used for static pages. I finally settled on using Google Firebase, which seemed perfect for what I hoped to accomplish. Firebase has built-in database support, this allowed me to store the code snippets in a JSON format, and dynamically display them on the website.
